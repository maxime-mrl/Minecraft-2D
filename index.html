Minecraft 2D

What is it?

Minecraft-2D is a showcase of procedural terrain generation using trigonometric functions. The terrain is generated randomly and infinitely using multiple sinus functions, this is inspired by Kevinverse work. The focus of the site is on showcasing the random generation of terrain, so the gameplay is really secondary just to give a "purpose" to the terrain.

How it works?

Terrain is generated with multiple sinus functions, by adding multiple of theses we get a curve that takes really long to reapeat and so feels totally random. This method is used for everything except monsters. Source code
